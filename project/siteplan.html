Site Plan for "We Stubborn Few"

1. Purpose and Goals

  • Provide information about the "We Stubborn Few" clan in Destiny 2.
  • Allow potential members to join the clan and connect with current members.
  • Showcase activities, dungeons, and raids in Destiny 2.
  • Facilitate communication and community building among members.

2. Target Audience

Players of Destiny 2 interested in joining a clan and current members seeking information and updates.

3. Site Structure

Home Page (project.html)

  • Overview of the clan.
  • Welcome message.
  • Navigation menu to other pages.
  • Container for displaying dungeons and activities.
  • Modal functionality for detailed views of dungeons/activities.

Join Fireteam Page (form.html)

  • Form for new members to submit their information.
  • Fields: Call Tag, Class, Power Level, SubClass.
  • Submission handling leading to a thank-you message page.

Fireteams Page (fireteam.html)

  • A list of current fireteams within the clan.
  • Information on how to join or participate in activities.

4. Content

Home Page

  • Welcome message and introduction to the clan.
  • Dynamic content loading for dungeons and activities using JSON data.

Join Fireteam Page

User-friendly form for data collection.

Fireteams Page

Display of active fireteams and their members.

5. Navigation

Header Navigation

Links to Home, Join Fireteam, and Fireteams pages.

Footer

Copyright notice and social media links.

6. Technical Specifications

HTML

Valid, standards-compliant HTML structure.

CSS

Consistent styling across all pages using external stylesheets.

JavaScript

Dynamic content loading from a JSON file.

7. SEO and Accessibility

  • Use of descriptive meta tags for SEO optimization.
  • Accessibility features like ARIA roles and properties.

8. Future Enhancements

  • Integration of a community forum or chat feature.
  • Calendar of upcoming events and raids.
