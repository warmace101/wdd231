We Stubborn Few Clan Site

A clan that tries to be an open and welcoming community. We strive to teach people how to do the endgame

Site Purpose

This site provides a place to ask questions about any endgame content, whether it be dungeons or raids. It also tries to help find groups within the clans to do endgame content with so that we earn clan xp to boost rewards from any activity a clan member completes.
